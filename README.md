## 📌 Описание

Этот скрипт предоставляет расширенный ESP (Extra Sensory Perception) для игры **Unknown Threat** на платформе **Roblox**. Он специально адаптирован для **мобильных устройств** (телефоны и планшеты), но также отлично работает на ПК.

Все настройки **автоматически сохраняются** в папку `ESP_Settings` внутри игрового мира (workspace) и загружаются при следующем запуске.

---

## 🎯 Возможности

### ESP (Extra Sensory Perception)

| Функция | Описание |
|---------|----------|
| **Box ESP** | Отображение рамки вокруг игрока (в стиле «Corner» или «Full») |
| **Skeleton ESP** | Отображение скелета игрока |
| **Name ESP** | Отображение никнейма над игроком |
| **Health Bar** | Полоса здоровья с возможностью отображения числового значения |
| **Tracer ESP** | Линия от указанной точки (низ, верх, центр, курсор) до игрока |
| **Head Point** | Точка над головой игрока |
| **Team Check** | Отключение ESP для союзников (на основе игровых ролей) |
| **Wall Check** | Скрытие ESP за стенами (не работает сквозь препятствия) |
| **Max Distance** | Настройка максимальной дистанции отображения ESP |
| **Rainbow Mode** | Динамическая смена цветов для всех игроков |

### 🎨 Настройка цветов ролей

Вы можете настроить цвета для каждой роли в игре:

- **Seeker / Killer** – красный
- **Hider** – жёлтый
- **Innocent** – зелёный
- **Traitor** – фиолетовый
- **Police / SWAT / Sheriff** – синий
- **Juggernaut** – оранжевый
- **No Role** – белый
- **Unknown** – серый

## 📱 Мобильная адаптация

Скрипт полностью оптимизирован для мобильных устройств:

- ✅ Адаптивный размер окна GUI (80% ширины, 60% высоты экрана)
- ✅ Убрана клавиша сворачивания (не работает на сенсорных экранах)
- ✅ Автоматическая замена режима трассера «Mouse» на «Center» при использовании сенсорного управления
